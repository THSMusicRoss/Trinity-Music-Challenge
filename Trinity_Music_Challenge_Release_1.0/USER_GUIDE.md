# Trinity Music Challenge — Teacher User Guide

## Starting a game

Choose **New Game**, then complete the five setup steps:

1. Select Manual Classroom Mode or Built-in Buzzers.
2. Set up two to six teams.
3. Select question categories, difficulty levels, question types, game length, and distribution.
4. Select scoring and gameplay rules.
5. Review the game and optionally save it as a preset.

## Manual Classroom Mode

Students may raise hands or use a physical classroom buzzer. Select the team that answered correctly.
Choose **No Correct Answer** when no team earns the question.

## Built-in Buzzer Mode

Each team has an on-screen buzzer. The first buzzer locks the others. The teacher judges the answer.
If steals are enabled, an incorrect team is locked out and the remaining teams may buzz.

Number keys **1–6** activate the corresponding team buzzers.

## Keyboard shortcuts

- **1–6:** Team buzzers
- **Space** or **R:** Reveal answer
- **N:** Next question
- **P:** Pause or resume
- **S:** Skip question
- **B:** Reset buzzers
- **Escape:** Return toward the main menu; active games receive a confirmation

## Scoring

- **Difficulty-Based:** Uses the Easy, Medium, and Challenging values.
- **Fixed:** Every question has the same value.
- **Custom:** The teacher chooses the value of each difficulty level.

Use **Adjust** below a team's score to make a manual correction.

## Saved games

The app autosaves during play. After a refresh or browser restart, choose **Resume Game**. A completed
game is removed from the resumable-game slot.

## Presets

Save a setup from the Review step. Use **Session Presets** on the main menu to load or delete one.

## Sound

Sound effects use the browser's audio system and require no downloaded sound files. Some browsers do
not allow audio until the teacher has clicked or pressed a key within the page.

## Projection tips

Use **Presentation Mode** for fullscreen display. For the clearest classroom view, use a modern version
of Chrome, Edge, Firefox, or Safari and set browser zoom near 100%.

## Browser storage warning

Clearing browser history or site data may erase saved games, presets, and preferences.
